# CV Tailoring System

Repository workspace to build and maintain tailored CVs, cover letters, and role-specific application material.

## Structure

- `master_profile/` full source material about your experience
- `target_roles/` role-focused positioning
- `companies/` company-specific customizations
- `templates/` reusable CV and cover-letter templates
- `job_descriptions/` pasted or saved job ads
- `generated_versions/` finalized tailored outputs
- `prompts/` prompts and workflows for adapting content

## Workflow

1. Keep your full experience updated in `master_profile/`
2. Maintain role-specific angles in `target_roles/`
3. Add a target company folder under `companies/`
4. Save each job description in `job_descriptions/`
5. Generate a tailored CV and cover letter into `generated_versions/`
